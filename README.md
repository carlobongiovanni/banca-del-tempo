# banca-del-tempo

This SPA is deployed into [bancadeltempo.netlify.app](https://bancadeltempo.netlify.app/) and is a volunteer project done for a friend. 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
